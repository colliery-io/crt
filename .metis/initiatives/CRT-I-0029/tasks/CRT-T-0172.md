## Implementation Notes

### Files to test
- `src/render/mod.rs` — `process_pty_updates`, `compute_shell_event_overrides`, `compute_effect_patches`
- `src/window.rs` — `prepare_render_cells` (1 smoke test exists)

### Test areas
1. **Shell event overrides**: Bell event triggers flash override, command exit success/fail triggers appropriate override, multiple events in single batch, events with no matching theme overrides
2. **Effect patches**: Override active → Apply patch, override expired → Restore patch, no overrides → empty patches, multiple simultaneous overrides
3. **Render cell preparation**: Default foreground/background colors, ANSI 16-color palette, 256-color, truecolor, bold/italic flags, URL hover underline decoration, search match highlighting, semantic zone coloring
4. **PTY update processing**: Uses MockPty — feed known output, verify content_changed flag, verify shell events extracted, verify title change detection

### Dependencies
- Requires constructing test `Theme` objects — check if `crt_theme` has test helpers or use default theme
- `prepare_render_cells` needs `RenderContext` with mock data — already demonstrated in smoke test
- `process_pty_updates` needs `ShellTerminalGeneric<MockPty>` from CRT-I-0028
