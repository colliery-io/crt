# Input handling edge case tests

## Parent Initiative

[[CRT-I-0029]]

## Objective

Extend the input handling test suite with edge cases and broader coverage. CRT-I-0028 added 11 keyboard tests and 16 mouse tests using pure functions (`determine_command_shortcut`, `determine_scroll_action`, `screen_to_grid_position`, `determine_click_target`, `compute_click_count`). This task adds coverage for key encoding, URL detection edge cases, and more modifier combinations.

## Acceptance Criteria

- [ ] Key encoder (`key_encoder.rs`) coverage expanded — currently 7 tests, add special key combos (Ctrl+arrows, Alt+keys, function keys F1-F12)
- [ ] URL detection edge cases — parentheses in URLs, trailing punctuation, query strings, fragments, `www.` prefix handling
- [ ] Mouse report protocol — SGR vs legacy encoding, edge coordinates (0,0 and max bounds)
- [ ] Multi-line URL merging (`merge_wrapped_urls`) — currently 0 tests
- [ ] All existing tests still pass

## Implementation Notes

### Files to test
- `src/input/key_encoder.rs` — Key-to-escape-sequence encoding (7 existing tests)
- `src/input/mod.rs` — URL detection, mouse reporting, `merge_wrapped_urls` (1 existing test)
- `src/input/keyboard.rs` — Pure shortcut functions (11 existing tests from CRT-I-0028)
- `src/input/mouse.rs` — Pure mouse functions (16 existing tests from CRT-I-0028)

### Test areas
1. **Key encoding**: F1-F12, Ctrl+letter (a-z), Alt+letter, Shift+special keys, numpad keys, Insert/Delete/PageUp/PageDown
2. **URL detection**: `https://example.com/path(with_parens)`, trailing `.` and `,`, `file://` URLs, bare `www.` domains, URLs with `#fragment` and `?query=val`
3. **URL merging**: URLs wrapping across lines at column boundary, non-continuation detection (next line starts with protocol), empty next line
4. **Mouse reporting**: `mouse_report()` for SGR mode vs legacy X10 mode, boundary coordinates (col=0, col=max), scroll button codes
5. **Additional keyboard shortcuts**: All Cmd+digit tab switching (1-9), zoom in/out/reset, search navigation (Cmd+G, Shift+Cmd+G)
