# Effects system and matrix rain tests

## Parent Initiative

[[CRT-I-0029]]

## Objective

Add tests for the effects system, particularly the untested `matrix.rs` and `renderer.rs` modules. The effects crate has 32 tests spread across 7 files, but `matrix.rs` (matrix rain effect) and `renderer.rs` (effect orchestration) have zero coverage.

## Acceptance Criteria

## Acceptance Criteria

## Acceptance Criteria

- [ ] 15+ new tests added to effects modules
- [ ] `matrix.rs` has tests for column initialization, character cycling, drop speed, and reset behavior
- [ ] `renderer.rs` has tests for multi-effect management (enable/disable/update lifecycle)
- [ ] Effect configuration application tested — verify `EffectConfig` parameters correctly change behavior
- [ ] All existing 32 effect tests still pass

## Implementation Notes

### Files to test
- `crates/crt-renderer/src/effects/matrix.rs` — Matrix rain effect (0 tests)
- `crates/crt-renderer/src/effects/renderer.rs` — Effect orchestration (0 tests)
- Existing tested files for reference: `motion.rs` (10), `grid.rs` (5), `shape.rs` (4), `starfield.rs` (4), `particles.rs` (3), `rain.rs` (3), `sprite.rs` (3)

### Test areas
1. **Matrix rain**: Column state initialization, character selection/cycling, drop speed variation, fade trail behavior, reset/clear
2. **Effect renderer lifecycle**: Add effect, update frame, remove effect, multiple effects simultaneously active, effect config hot-swap
3. **Config application**: Verify that changing `EffectConfig` fields (speed, density, color) produces expected state changes on next update
4. **Edge cases**: Zero-size viewport, very large viewport, effect update with dt=0, rapid enable/disable toggling

### Approach
- Read existing effect tests (e.g., `starfield.rs`, `motion.rs`) to understand test patterns used in the crate
- Matrix rain likely uses internal state (columns, characters, positions) — test state transitions rather than visual output
- `renderer.rs` may need MockRenderer or similar — check if `mock.rs` (15 tests) provides helpers
