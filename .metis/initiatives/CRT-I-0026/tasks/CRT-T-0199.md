# Tab merging into existing window

## Objective

Implement tab merging: dragging a tab onto another window's tab bar inserts it there. This reuses the tab/shell extraction from CRT-T-0198 but targets an existing window instead of creating a new one.

### User interaction
1. User drags a tab over another window's tab bar
2. An insertion caret shows where the tab will land in the target bar
3. User releases → tab moves from source window to target window at the indicated position
4. Shell session continues uninterrupted in the target window

## Acceptance Criteria

- [ ] The merged tab becomes the active tab in the target window

- [ ] `content_hashes` entry transfers from source to target
- [ ] No tab ID collisions (guaranteed by global IDs from CRT-T-0194)
- [ ] Edge case: dragging onto same window's tab bar falls through to reorder (not merge)

## Implementation Notes

### Merge execution (src/app/handler.rs or new src/app/tab_transfer.rs)

```rust
fn merge_tab(
    &mut self,
    tab_id: TabId,
    source_window_id: WindowId,
    target_window_id: WindowId,
    insert_index: usize,
) {
    // 1. Extract from source
    let source = self.windows.get_mut(&source_window_id).unwrap();
    let tab = source.gpu.tab_bar.remove_tab(tab_id).unwrap();
    let shell = source.shells.remove(&tab_id).unwrap();
    let content_hash = source.content_hashes.remove(&tab_id).unwrap_or(0);

    // 2. Insert into target
    let target = self.windows.get_mut(&target_window_id).unwrap();
    target.gpu.tab_bar.insert_existing_tab(tab, insert_index);
    target.shells.insert(tab_id, shell);
    target.content_hashes.insert(tab_id, content_hash);
    target.gpu.tab_bar.select_tab(tab_id);

    // 3. Resize shell to match target
    let size = Size::new(target.cols, target.rows);
    target.shells.get_mut(&tab_id).unwrap().resize(size);
}
```

### Insert at index

`TabBarState` needs `insert_existing_tab(tab, index)` (not just `add_existing_tab` which appends). Or rename the method to be index-aware.

### Shared extraction helper

Both detach (CRT-T-0198) and merge extract a tab+shell from a source window. Factor this into a helper:

```rust
fn extract_tab(window: &mut WindowState, tab_id: TabId) -> Option<(Tab, ShellTerminal, u64)> {
    let tab = window.gpu.tab_bar.remove_tab(tab_id)?;
    let shell = window.shells.remove(&tab_id)?;
    let hash = window.content_hashes.remove(&tab_id).unwrap_or(0);
    Some((tab, shell, hash))
}
```
