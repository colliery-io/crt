### 2025-11-27: Implementation Complete

Implemented right-click context menu with the following features:

**Files Modified:**
- `src/window.rs` - Added `ContextMenuItem` enum and `ContextMenu` struct
- `src/main.rs` - Added event handling and `handle_context_menu_action` function
- `src/render.rs` - Added `render_context_menu` function

**Features Implemented:**
- Right-click anywhere shows context menu at cursor position
- Menu items: Copy, Paste, Select All with keyboard shortcut hints
- Hover highlighting on menu items
- Click on item executes action (Copy/Paste/Select All)
- Click outside menu or press Escape dismisses menu
- Menu automatically adjusts position to stay within screen bounds
- Dark semi-transparent background with border styling
