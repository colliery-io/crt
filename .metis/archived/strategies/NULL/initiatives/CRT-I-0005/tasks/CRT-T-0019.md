### Property Mapping

| Category | CSS Properties | Shader/Renderer Mapping |
|----------|---------------|------------------------|
| Typography | `font-family`, `font-size`, `line-height`, `font-feature-settings` | glyphon FontSystem, Metrics |
| Font variants | `--font-bold`, `--font-italic`, `--font-bold-italic` | Font loading |
| Colors | `color`, `background`, `--color-*` | Uniforms, ANSI palette |
| Selection | `::selection { background, color }` | Selection render state |
| Highlight | `::highlight { background, color }` | Search match render state |
| Effects | `text-shadow`, `--grid-*` | Glow/grid shader uniforms |
| Cursor | `::cursor { background }` | Cursor color uniform |

### Color Support

- **16 ANSI colors**: Customizable via theme
- **256 colors**: Standard palette fallback
- **True color (24-bit)**: Pass-through from terminal apps

### Technical Approach

1. Build Theme struct matching CSS properties
2. Implement to_uniforms() for GPU buffer generation
3. Create EffectPipeline driven by theme properties
4. Effects enabled based on which properties are set
