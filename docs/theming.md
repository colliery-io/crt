# CRT Terminal Theming

CRT Terminal uses CSS-like syntax for theming. Themes are stored in `~/.config/crt/themes/` and selected via `config.toml`.

## Documentation

- **New to theming?** Start with the [Build Your First Theme](tutorials/build-your-first-theme.md) tutorial
- **Quick setup:** [How to Create a Custom Theme](how-to/create-custom-theme.md)
- **All CSS properties:** [Theme CSS Properties Reference](reference/theme-css-properties.md)
- **Built-in themes:** [Built-in Themes Reference](reference/built-in-themes.md) and [Theme Gallery](gallery/README.md)
- **Animated sprites:** [How to Create a Sprite Sheet from a GIF](how-to/create-sprite-from-gif.md)
- **Reactive events:** [How to Set Up Reactive Themes](how-to/set-up-reactive-themes.md)
